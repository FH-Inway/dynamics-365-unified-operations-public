# License summary report

[!include[banner](../../../finance/includes/banner.md)]
This report helps to understand how many Entry points on each license level are applied to User, Role, Duty and Privilege This helps to find some possible optimization within the organization.

## Recalculate prices
Recalculate prices allows you to recalculate the prices of the licenses which you use for Users, Roles, Duties and Privileges.

## Rebuild licenses
Rebuild licenses allows you to rebuild the licenses which you use for Users, Roles, Duties and Privileges. Potentially, all users with a small number of **Entry points** with the Operational license level might be moved into Activity or Team Member level just by redesigning (their access).

## User license
A list of all the users with all their details regarding licenses, operations, activities, etc This allows you to check each user’s access and verify what license level they have.

## Role license
A list of all the **Roles** and their details regarding licenses, operations, activities, etc This allows you to check each role accesses and verify what license level they have.

## Duty license
A list of all **Duties** and their details regarding licenses, operations, activities, etc. This allows you to check each duty’s accesses and verify what license level they have.

## Privilege license
A list of all the **Privileges** and their details regarding licenses, operations, activities, etc. This allows us to check each privilege’s accesses and verify what license they have.

## User license summary
A report that allows you to analyze your user licenses and verify all the details needed to assign licenses and optimize them for their usability in your work.

## Security summary
A report that allows you to analyze user licenses, Roles, Duties, and Privileges in order to optimize the environment so that each user or a group of users is granted appropriate access.
